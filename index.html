Key Features

  • Uses popular Adafruit ST7789 and XPT2046 libraries
  • Compatible with Arduino UNO family (tested on UNO R3)
  • Supports a wide range of touchscreen displays
  • Easy to extend and customize via GitHub
  • Open source and beginner-friendly

About the Project

ZaurOS is a demonstration operating system for Arduino boards, designed to showcase the use of modern TFT touchscreens and SD card storage. The project is built with the Arduino IDE and leverages well-known libraries for graphics and touch input. It is a great starting point for learning about embedded GUIs and hardware integration.
